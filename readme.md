# servalope
A simple django-based mailing app using an SQLite database.

## Requirements
- Python 2.7
- Django 1.10

## How To

Install Python and Django (Installation guide: https://docs.djangoproject.com/en/1.10/intro/install/), then use the following command at the root of this repo to start a webserver:

    $ python manage.py runserver

Navigate to localhost:8000 in a web browser to reach the app.
